Fracton order: sub-extensive degeneracy + immobile excitations. The X-cube model stores more logical qubits as it grows and its excitations cannot move (no string operator) — protection by immobility, the containment principle made physical. Anti-wash baseline: the 3D toric code (constant degeneracy, mobile excitations) on the same lattice. No φ.

The upper chart is the headline: as the lattice size L grows, the X-cube logical-qubit count k = 6L − 3 rises while the 3D toric code stays flat at k = 3. The lower chart measures mobility: a straight Z-string excitation in the 3D toric code costs a constant syndrome weight of 2 (the endpoints are mobile point charges joined by a string operator), whereas in the X-cube model the same string lights up syndrome weight 4·len — there is no string operator, so the excitation is frozen in place.
