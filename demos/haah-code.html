What this shows — the strongest fracton order, validated by its algebra

Haah's cubic code is type-II fracton order — the strongest kind. Every logical operator is fractal (Sierpinski-like): there are no string operators at all, so no excitation can move, even along a line or plane. Its ground-state degeneracy is famously irregular and number-theoretic in the system size L — which is exactly why a clean validation was hard. No φ: fracton order is φ-free stabilizer physics (the containment pole).

The resolution is to validate the clean algebra, not the formula-less degeneracy. CSS commutativity (defect 0) confirms it's a valid code.
